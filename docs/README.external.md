# TNT Collection

TNT Collection is a user-friendly toolkit for Ikariam players who want better city management, faster planning, and fewer manual clicks.

## 🎯 What it does for you

- Global city snapshot (resources, populations, income, and active buildings)
- Smart city switching with one-click navigation
- Production and consumption forecasts for all resource types
- Construction alerts and color-coded city status (building or idle)
- Optional premium-menu cleanup and UI improvements

## 🚀 Quick start (end-user)

1. Install a userscript manager: Tampermonkey / Violentmonkey / Greasemonkey
2. Add this script to the manager: `tnt.collection.user.js` (or latest from repo release)
3. Open Ikariam and login. An update bar appears with the TNT icon in the top navigation.
4. Click **TNT Options** to enable/disable features and personalize display settings.

## 🧰 Main features (in-game)

- **Resource Dashboard**: see per-city wood, wine, marble, crystal, sulphur, and population at a glance
- **Building Tracker**: track current building levels and a quick view of where construction is ongoing
- **Auto Switch**: switch through all your cities to update the resource and building overviews
- **Alerts**: show low resources, full warehouses warning signs
